# vue-authplugin
它是一个用于前端权限验证权限的Vue插件(通常用于验证登录下发的权限集合数组)，核心代码主要使用es6实现，可直接移植到其他框架中。

[English](./README_EN.md)

## 安装
使用npm：
```
$ npm install vue-authplugin
```
通过浏览器：
```
<script src="lib/vue-authplugin.js"></script>
```

## 开发
- 开发：`npm run dev`
- 编译：`npm run build`
- 打包umd:`npm run build:umd`
